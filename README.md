# CoreUI - Developer Analytics Dashboard 🚀

A sleek, responsive, and modern web-based dashboard designed for developers and tech leads to monitor financial and system metrics. This project features a native **Dark Mode** toggle and is localized for **Indian Rupee (₹)** denominations.

## ✨ Features

- **Dynamic Dark Mode:** Seamless transition between Light and Dark themes using CSS variables.
- **Responsive Design:** Optimized for desktop, tablet, and mobile viewing using CSS Grid and Media Queries.
- **Modern UI/UX:** Built with the 'Poppins' typeface and a glassmorphism-inspired sidebar.
- **Metric Cards:** Visualized cards for Revenue, Cloud Expenses, API Traffic, and User Licenses.

## 🛠️ Built With

*   **HTML5** - Semantic structure.
*   **CSS3** - Custom properties (variables), Grid, and Flexbox for layout.
*   **JavaScript (ES6)** - Theme switching logic and DOM manipulation.
*   **Google Fonts** - Poppins for professional typography.
